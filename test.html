Welcome to PAN-OS 7.1!


This release includes many new features, including this message of the day.
Some highlights of this release include:
  • SaaS Application Usage Report for visibility into sanctioned and unsanctioned SaaS applications running on your network
  • Support for large-scale, distributed User-ID deployments
  • AutoFocus threat intelligence integration with PAN-OS logs
  • Unified log view showing all traffic and threat-related logs on a single page
  • WildFire five-minute updates for detecting and blocking zero-day malware and exploits within minutes of discovery
  • Dynamic blocking of URLs in addition to IP addresses
  • Support for Amazon Elastic Load Balancing (ELB) on VM-Series firewalls
  • Simplified deployment of two-factor authentication on GlobalProtect

For a complete description of the new features and instructions on how to use them, refer to the PAN-OS 7.1 New Features Guide.

To ensure that you can take advantage of all of the new features, you must upgrade the Content Release version. Refer to the PAN-OS 7.1 Release Notes for the content version required.
